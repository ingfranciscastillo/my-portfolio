## Diseño

- **Tipografía**: PT Serif para texto
- **Paleta**: Azul primario, cyan, teal — colores cohesivos que transmiten confianza y profesionalismo
- **Cards con shadow pop**: Elementos que se elevan al hover, creando profundidad y respuesta visual
- **Modo oscuro**: Soporte nativo que ajusta la paleta sin perder identidad
- **Animaciones**: Transiciones sutiles que mejoran la experiencia sin distraer
